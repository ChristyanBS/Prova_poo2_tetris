🧪 COMO TESTAR TODAS AS VERSÕES
═══════════════════════════════════════════════════════════════════════════════

Opção 1: Script Automático (Recomendado)
─────────────────────────────────────────

   Windows:
   .\test-all-versions.bat

   Linux/Mac:
   chmod +x test-all-versions.sh
   ./test-all-versions.sh

   Menu Interativo com Opções:
   ├── 1) Compilar todas as versões
   ├── 2) Testar todas as versões
   ├── 3) Gerar Javadoc
   ├── 4) Executar jogo
   ├── 5) Build completo
   ├── 6) Info Git
   └── 7) Sair


Opção 2: Comandos Manuais
─────────────────────────

   # Compilar v0.1
   git checkout v0.1
   mvn clean compile

   # Compilar v0.2
   git checkout v0.2
   mvn clean compile

   # Compilar v1.0
   git checkout main
   mvn clean compile

   # Testar qualquer versão
   mvn test

   # Executar jogo (v1.0)
   git checkout main
   mvn exec:java -Dexec.mainClass="com.tetris.TetrisGame"

   # Gerar Javadoc
   mvn javadoc:javadoc
